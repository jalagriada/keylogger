python3 -m venv keylogger_venv - optional if you haven't installed venv or the virtual environment

source keylogger_venv/bin/activate - recommended prompt to execute after installing the venv


Guide:

1. Install the python3 -m venv keylogger_venv
2. Type source keylogger_venv/bin/activate
3. pip install -r requirements.txt
4. ls
5. python detector.py
6. python keylogger.py
7. python decoder.py "if you want to decode the keystrokes.txt for you to read properly.
